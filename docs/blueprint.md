# **App Name**: SourceClone

## Core Features:

- URL Input: Accepts a URL from the user.
- Site Fetching: Fetches the HTML, CSS, and JavaScript source code from the provided URL.
- Source Code Display: Displays the fetched source code (HTML, CSS, JavaScript) in a formatted and readable manner, allowing users to view each component separately.
- Download Functionality: Provides an option for users to download the complete source code as a ZIP file.

## Style Guidelines:

- Primary color: Dark blue (#2E3192) to give a professional, technical vibe.
- Background color: Light gray (#F0F0F0), a desaturated version of the primary hue.
- Accent color: Teal (#008080) to provide a contrast with the dark blue.
- Font pairing: Use 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Code Font: Use 'Source Code Pro' for display of the scraped source code.
- Simple, line-based icons to represent actions like 'download,' 'copy,' and different file types.
- Clean and structured layout with clear separation of code sections (HTML, CSS, JS).
